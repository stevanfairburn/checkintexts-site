# Checkintexts public site

This folder contains the public support, privacy, and terms pages required for Apple App Review.

## Preview locally
Open `public-site/index.html` in a browser.

## Publish with GitHub Pages
1. Push this repo to GitHub.
2. In the repo settings, go to Pages.
3. Set Source to "Deploy from a branch" and choose the `main` branch.
4. Set the folder to `/public-site`.
5. Save and wait for the Pages URL to become active.

The custom domain is configured via `public-site/CNAME`.
